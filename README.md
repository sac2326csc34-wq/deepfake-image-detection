# Deepfake Image Detection

This project detects whether an image is **real or fake (deepfake)** using a **Convolutional Neural Network (CNN)** model.  
The system allows users to upload an image through a web interface and predicts whether the image is authentic or manipulated.

---

## Technologies Used
- Python
- Flask
- TensorFlow / Keras
- HTML
- CSS

---

## Project Features
- Upload an image
- Detect whether the image is real or fake
- Simple web interface
- CNN-based deep learning model

---

## Project Structure

deepfake-image-detection

## How to Run the Project
1. Clone the repository
  git clone https://github.com/sac2326csc34-wq/deepfake-image-detection.git
   
2. Install required libraries
  pip install flask tensorflow keras opencv-python numpy

3. Run the application
   python app.py

4. Open browser and go to

http://127.0.0.1:5000
